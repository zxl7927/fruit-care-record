果蔬施肥记录 - 免费网站发布包

文件说明：
- index.html：网站首页，已由你的 HTML 文件改名而来
- manifest.webmanifest：让 iPhone 可以“添加到主屏幕”
- icon.svg / icon-192.png / icon-512.png / apple-touch-icon.png：网站图标
- sw.js：简单离线缓存文件，上线后可提高打开速度

GitHub Pages 免费发布步骤：
1. 打开 github.com 并登录。
2. 新建仓库，例如 fruit-care-record。
3. 上传本文件夹里的所有文件，不要只上传 ZIP。
4. 进入仓库 Settings → Pages。
5. Source 选择 Deploy from a branch，Branch 选择 main / root，保存。
6. 等待 1-3 分钟，得到类似：
   https://你的用户名.github.io/fruit-care-record/
7. 用 iPhone Safari 打开网址，点分享按钮 → 添加到主屏幕。

注意：
- 记录数据保存在浏览器本地 localStorage。换手机、清浏览器数据会丢失，所以请经常点网页里的“备份”。
- 如果网页改版后手机没变化，刷新网页或删除主屏幕图标后重新添加。
